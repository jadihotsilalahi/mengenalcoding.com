MENGENAL CODING

Apa Itu Coding ?

Coding adalah kegiatan menulis sekumpulan kode untuk berkomunikasi dengan komputer. Dengan kode itu, kita dapat meminta komputer untuk menjalankan suatu hal sesuai instruksinya. Misalnya, membuat website, menyusun aplikasi, memproses data, atau hal yang berkaitan dengan teknologi lainnya.

Apa Fungsi Coding ?

coding itub membantu komunikasi dengan komputer jauh lebih efektif dan bisa menjalankan perintah yang kalian inginkan . coding sebagai cara agar bisa berkomunikasi dengan komputer lebih maksimal terutama untuk memberikan perintah tertentu untuk mengerjakan sesuatu.

Apa Manfaat Belajar Coding?

Dengan belajar coding dapat membuat kamu terbiasa dalam pemecahan masalah dan mencaari langkah -langkah logis untuk membuat sebuah program bisa bekerja.Selain itu , Belajar coding juga akan membuatmu melihat masalah sebagai tantangan untuk berkembang,bukan sebagai halangan.

Meski kelihatannya rumit, ada banyak manfaat yang bisa kamu dapat dengan belajar coding. Ini adalah beberapa di antaranya:

1.Punya banyak pilihan Karir
2.Memulai Berbisnis sendiri
3.Pekerjaan Yang Paling Banyak Di Cari Oleh Perusahaan

Macam-Macam Bahasa Pemograman:

1.HTML

Secara umum, fungsi HTML adalah untuk mengelola serangkaian data dan informasi sehingga suatu dokumen dapat diakses dan ditampilkan di Internet melalui layanan web.

2.CSS

CSS berfungsi untuk membantu para web designer agar dapat mengubah dan menambahkan, baik teks, gambar, hingga latar belakang sebuah halaman HTML. Biasanya, keberadaan CSS dapat langsung terlihat dengan adanya perbedaan pada warna teks.

3.Java Script

JavaScript adalah bahasa pemrograman yang digunakan developer untuk membuat halaman web yang interaktif.

4.Python

Python dapat digunakan untuk pengembangan web (sisi server), pengembangan perangkat lunak atau membuat aplikasi (software). Python dapat membantu menyelesaikan persamaan matematika, pembuatan skrip sistem dan pemrograman mikrokontroller (MicroPython).
